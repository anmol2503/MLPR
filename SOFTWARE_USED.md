# Software Used

## Operating Environment

- Operating System: macOS
- Programming Language: Python 3
- Version Control System: Git
- Repository Hosting Platform: GitHub

## Development Tools

- Jupyter Notebook: used for exploratory data analysis, preprocessing, feature engineering, and model experimentation
- Visual Studio Code / Code Editor: used for writing and editing project files
- Terminal / Command Line: used for environment setup, dependency installation, Git operations, and project execution

## Python Libraries Used

- `pandas`: data cleaning, preprocessing, and tabular data handling
- `numpy`: numerical operations
- `matplotlib`: data visualization and chart generation
- `scikit-learn`: regression models, preprocessing utilities, and evaluation workflows
- `statsmodels`: Poisson regression and statistical modeling
- `joblib`: saving and loading trained machine learning models
- `gradio`: deployment of the final interactive prediction web application

## Machine Learning / Modeling Tools

- Poisson Regression: final deployed model for cafeteria footfall prediction
- Linear Regression: baseline model experimentation
- Ridge Regression: regularized regression experimentation
- Lasso Regression: feature-sensitive regression experimentation
- ElasticNet: combined regularization experimentation
- Decision Tree Regressor: non-linear regression experimentation
- Random Forest Regressor: ensemble-based regression experimentation
- Gradient Boosting Regressor: boosted ensemble regression experimentation
- Support Vector Regressor (SVR): alternative regression experimentation
- Keras / Neural Network models: deep learning-based experimental models

## Data Files and Artifacts

- CSV files: used to store cleaned and processed datasets
- Jupyter notebooks (`.ipynb`): used to document experiments and model development
- Pickle model file (`poissonreg.pkl`): stores the trained final model used in the app

## Deployment and Output

- Gradio Web Application: final interface for entering inputs and viewing predictions
- Browser-based Local Host: used to access the application at `http://127.0.0.1:7860`

## Summary

This project was developed using Python-based data science and machine learning tools, with Jupyter Notebooks for experimentation, Git and GitHub for version control, and Gradio for deployment of the final cafeteria footfall prediction system.
